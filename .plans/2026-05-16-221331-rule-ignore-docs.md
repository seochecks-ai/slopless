# Goal

Tell users how to ignore specific Slopless rules, and make the documented ignore comments work in the wrapper.

# Approach

- Add `textlint-filter-rule-comments` as a runtime dependency.
- Ship a small Slopless textlint config that enables the comments filter.
- Load that config by default in the Slopless CLI, unless the user explicitly passes `--config`.
- Document rule-specific, multi-rule, and all-rule ignore blocks in README and CLI help.
- Verify ignored blocks with real CLI output.
