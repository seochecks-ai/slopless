# Summary

Published `slopless@0.2.2` from the standalone Slopless repository.

The npm package now points at `agent-quality-controls/Slopless` and installs as a self-contained JSON-only CLI.

# Decisions Made

- Released a patch version because the package code is functionally the same CLI, but its source repository and package metadata changed.
- Verified both the packed tarball and the registry install before treating the release as complete.

# Key Files For Context

- `package.json`
- `package-lock.json`
- `README.md`
- `src/cli.ts`

# Verification

- `npm view slopless version dist-tags.latest --json` showed `0.2.1` before publish.
- `npm pack --json` produced `slopless-0.2.2.tgz`.
- Fresh tarball install: `npx slopless --help`, `npx slopless --version`, and `npx slopless sample.md` worked. The sample command emitted parseable JSON and exited 1 because it found one prose issue.
- `npm publish --dry-run --access public`.
- `npm publish --access public`.
- `npm view slopless versions version dist-tags.latest repository.url homepage bugs.url --json` showed `0.2.2` and the new GitHub URLs.
- Fresh registry install of `slopless@0.2.2`: `npx slopless --version` printed `0.2.2`, and `npx slopless sample.md` emitted parseable JSON with one finding.
