# GCA - Words To Watch

Source: https://www.gca.gov.uk/government-commercial-agency-style-guide/language/words-to-watch

| Term | Guidance | Alternatives |
|---|---|---|
| Advance | Be more specific to the context you are writing about for better clarity | Improve / make progress / help / come / lead |
| Agenda | Only use this word when referring to an agenda meeting | Plan |
| Deliver | Only use this word in a technical context. Such as Strategic delivery or contract delivery. In all other contexts, be more specific to the context you are writing about. | Make / create / provide / achieve |
| Drive | Unless you are referring to driving a vehicle, always try to use words that are more specific to the context you are writing about. | Move forward / push / urge |
| Collaborate | There are simpler alternatives | Work with |
| Combat | Unless you are referring to military, be more specific with your context | Solve / fix |
| In order to | – | to |
| Just | Not needed, filler word. Remove and avoid. | N/A |
| Savings | Do not use this word when discussing the benefits or value we add as we provide a range of benefits. | Commercial benefits |
| Simple | Saying something is ‘simple’ is subjective. What is simple for one person may not be simple for someone else. It is better to remove this word as it adds no further context | N/A |
| Transformation | Only use this word when talking about transforming public procurement or digital transformation | Change / changing / evolve |
| Web pages / Web page | This should be one word | Webpages / webpage |
