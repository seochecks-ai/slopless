### Signs that text was written by ChatGPT
- Complex sentence structures
- Tired and meaningless statements
- An unusually formal tone in text that’s supposed to be conversational or casual — or, an overly casual tone for a text that’s supposed to be formal or business casual
- Unnecessarily long and wordy
- Vague statements


### Useful prompts

**Case** Your answering a comment in your linkedin post:
Try to even so guide the gpt on what you want
```
based on this comment ....xxx... answer being polite but avoiding the use of common words, make it simple and base on this comment I have: ....xxxx....
```

**Case** writing an article about Rust programming language:
```
Write a detailed and informative article about the Rust programming language. Avoid using the words: Delve, Dive, Explore, Aesthetic, Seamlessly, Realm, World, Illustrious, Unwavering, Additionally, Tapestry, Saga, Beacon, Domain, Ingenuity, Grand, Steadfast, Crucible, Unraveling, Resistance, Myriad, Unfathomable, Solace, Landscape, Legacy, Unveiling, Unveiled, Pivotal. Instead, use clear, concise, and technical language to describe Rust’s features, advantages, and real-world applications. Focus on Rust’s safety, concurrency, performance, and community-driven development. Use concrete examples and comparisons with other languages like C++ and Python to illustrate Rust’s strengths.
```
